## Quick Start

```tsx
import { Button } from '@pineapple-ui/core';

function App() {
  return (
    <Button variant="primary" size="md">
      Click me
    </Button>
  );
}
```

## Packages

| Package | Description |
|---------|-------------|
| `@pineapple-ui/core` | Styled components with CVA variants |
| `@pineapple-ui/primitives` | Headless accessible components |
| `@pineapple-ui/tokens` | Design tokens and theme |
| `@pineapple-ui/icons` | SVG icon components |
| `@pineapple-ui/utils` | Utility functions |
