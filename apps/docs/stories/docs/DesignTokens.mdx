# Design Tokens

Pineapple UI uses CSS custom properties (design tokens) for consistent theming across all components.

## Colors

All colors use the `oklch` color space for perceptually uniform color manipulation.

### Primary

| Token | Value | Usage |
|-------|-------|-------|
| `--color-primary-50` | `oklch(0.97 0.02 250)` | Lightest background |
| `--color-primary-100` | `oklch(0.94 0.04 250)` | Light background |
| `--color-primary-500` | `oklch(0.60 0.20 250)` | Main brand color |
| `--color-primary-600` | `oklch(0.52 0.18 250)` | Hover state |
| `--color-primary-700` | `oklch(0.44 0.16 250)` | Active state |

### Semantic Colors

| Scale | Usage |
|-------|-------|
| `success` | Positive actions, confirmations |
| `warning` | Caution, attention needed |
| `danger` | Errors, destructive actions |
| `neutral` | Text, borders, backgrounds |

## Spacing

Based on a 4px base unit for consistent rhythm.

| Token | Value | Usage |
|-------|-------|-------|
| `--spacing-0` | `0px` | No spacing |
| `--spacing-1` | `4px` | Tight spacing |
| `--spacing-2` | `8px` | Small spacing |
| `--spacing-4` | `16px` | Default spacing |
| `--spacing-6` | `24px` | Medium spacing |
| `--spacing-8` | `32px` | Large spacing |

| Token | Value |
|-------|-------|
| `--font-sans` | `'Inter', system-ui, sans-serif` |
| `--font-mono` | `'JetBrains Mono', monospace` |

| Token | Value |

## Shadows

| Token | Value |
|-------|-------|
| `--shadow-sm` | `0 1px 2px 0 rgb(0 0 0 / 0.05)` |

| `--shadow-lg` | `0 10px 15px -3px rgb(0 0 0 / 0.1)` |

## Dark Mode

Toggle dark mode by adding the `dark` class to a parent element:

```html
<html class="dark">
  <!-- Components will use dark mode tokens -->
</html>
```

Or use the CSS media query:

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* Dark mode tokens applied automatically */
  }
}
```

## Customization

Override tokens in your CSS:

```css
@theme {
  --color-primary-500: oklch(0.65 0.25 200);
  --font-sans: 'Your Font', sans-serif;
}
```

Or use Tailwind config:

```js
// tailwind.config.js
export default {
  theme: {
    extend: {
      colors: {
        primary: {
          500: 'oklch(0.65 0.25 200)',
        },
      },
    },
  },
};
```
